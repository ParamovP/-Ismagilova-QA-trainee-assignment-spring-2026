# Bug Reports

## BUG-01: Фильтр максимальной цены допускает значения выше указанного значения

**Приоритет:** P2    
**Окружение:** OC Windows 10 Pro, Chrome 146.0.7680.81 

**Шаги для воспроизведения:**
1. Открыть [сайт](https://cerulean-praline-8e5aa6.netlify.app/) 
2. Ввести в поле "От" значение "2000"
3. Ввести в поле "До" значение "16000"
4. Проверить цены объявлений в выдаче

**Фактический результат:**
- В результатах отображаются объявления с ценой выше 16000
Автотест:
- Файл: `categoryFilterCheck.test.ts`
- Тест: `PD-01: Базовая проверка`

**Ожидаемый результат:**
- Отображаются только объявления с ценой в диапазоне 2000–16000

## BUG-02: Включение тогла "Только срочные" не отфильтровывает объявления без отметки "Срочно"

**Приоритет:** P2    
**Окружение:** OC Windows 10 Pro, Chrome 146.0.7680.81 

**Шаги для воспроизведения:**
1. Открыть [сайт](https://cerulean-praline-8e5aa6.netlify.app/)
2. Переключить тогл "Только срочные" в активное состояние
3. Проверить выдачу объявлений 

**Фактический результат:**
- результаты выдачи не меняются, отображаются объявления без отметки "Срочно"
Автотест:
- Файл: `onlyUrgentCheck.test.ts`
- Тест: `Проверка работы тогла «Только срочные»`

**Ожидаемый результат:**
- в выдаче отображаются только объявления со знаком "Срочно"

## BUG-03: Не работает кнопка запуска таймера после его остановки на странице Статистика

**Приоритет:** P3    
**Окружение:** OC Windows 10 Pro, Chrome 146.0.7680.81 

**Шаги для воспроизведения:**
1. Открыть [сайт](https://cerulean-praline-8e5aa6.netlify.app/)
2. Перейти на страницу статистики нажатием на кнопку "Статистика" в хэдере страницы
3. Нажать кнопку "Отключить автообновление"
4. Нажать на появившуюся вместо неё кнопку "Включить автообновление"

**Фактический результат:**
- при нажатии на кнопку "Включить автообновление" ничего не происходит
Автотест:
- Файл: `refreshTimer.test.ts`
- Тест: `Проверка работы кнопки остановки и запуска таймера`

**Ожидаемый результат:**
- при нажатии на кнопку "Включить автообновление" возвращается отображение таймера и полоски отсчета
