# Test Cases

## TC-01: Проверка работы фильтров на диапазон цены с корректными данными

**Шаги:**
1. Открыть [сайт](https://cerulean-praline-8e5aa6.netlify.app/)
2. Заполнить поля "От" и "До" в области фильтров цены корректными данными
3. Проверить цены в выдаче на соответствие указанному диапазону

**Ожидаемый результат:**
- в выдаче отображаются только объявления с ценой в пределах указанного диапазона

**Тестовые данные**
- "От" = 2000, "До" = 16000     - базовая проверка
- "От" = 0, "До" = 9999999      - минимум = 0 и максимум = большое число
- "От" = 6343.5, "До" = 29050.3 - дробные числа
- "От" = 2000, "До" пустое      - только "От"
- "От" пустое, "До" = 16000     - только "До"

## TC-08: Проверка переключения между тёмной и светлой темой на мобильной версии сайта 

**Шаги:**
1. Открыть [сайт](https://cerulean-praline-8e5aa6.netlify.app/)
2. Переключиться в режим мобильных устройств в DevTools (toggle device toolbar)
3. Нажать на кнопку с изображением месяца в правом верхнем углу страницы и проверить стиль темы оформления
4. Нажать на кнопку с изображением солнца и провериить стиль темы оформления

**Ожидаемый результат:**
- по умолчанию выбрана светлая тема оформления
- при нажатии на значок месяца тема меняется на темную, а значок месяца меняется на значок солнца
- при нажатии на значок солнца тема меняется на светлую, а значок солнца меняется на значок луны
